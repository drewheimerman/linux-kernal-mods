README

COMPILATION:

Follow standard gcc compilation procedures for each .c file.

EXECUTION:

—MUST RUN PROCESS 0 FIRST—

Process 0 - './proc0'
Process 1 (Reader) - './reader' to enter input from stdin, or './reader < inputFile' to read from a file
Process 2 (Modifier) - './modifier'
Process 3 (Printer) - './printer'

OUTPUT:

The output of the program will be located in "output.txt". The data from each run will be appended onto the same file (does not overwrite previous outputs).
